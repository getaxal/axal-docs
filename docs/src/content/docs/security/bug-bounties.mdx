# Bug Bounties

Help secure Axal by participating in our bug bounty program.

## Program Overview

We offer competitive rewards for security researchers who identify vulnerabilities in our platform.

## Scope

### In Scope
- **Smart Contracts**: All deployed smart contracts
- **Web Application**: Main website and dashboard
- **Mobile Apps**: iOS and Android applications
- **API Endpoints**: All public API endpoints
- **Infrastructure**: Cloud infrastructure and services

### Out of Scope
- **Third-party Services**: Services not directly controlled by Axal
- **Social Engineering**: Phishing or social engineering attacks
- **Physical Security**: Physical access to facilities
- **Denial of Service**: DDoS attacks or service disruption

## Reward Tiers

### Critical ($10,000 - $50,000)
- Remote code execution
- Private key compromise
- Fund theft or manipulation
- Critical smart contract vulnerabilities

### High ($5,000 - $10,000)
- SQL injection
- Cross-site scripting (XSS)
- Authentication bypass
- Significant data exposure

### Medium ($1,000 - $5,000)
- Information disclosure
- Privilege escalation
- Business logic flaws
- Moderate data exposure

### Low ($100 - $1,000)
- Minor information disclosure
- UI/UX security issues
- Best practice violations
- Low-impact vulnerabilities

## Submission Process

1. **Discovery**: Identify a security vulnerability
2. **Documentation**: Create detailed proof of concept
3. **Submission**: Submit via our security portal
4. **Review**: Our security team reviews the submission
5. **Reward**: Receive payment upon confirmation

## Responsible Disclosure

- **No Public Disclosure**: Keep findings private until resolved
- **Reasonable Timeline**: Allow time for fixes to be implemented
- **Coordinated Release**: Work with our team on disclosure
- **No Exploitation**: Do not exploit vulnerabilities beyond testing
