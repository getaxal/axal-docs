# Self-Custodied Wallets

Understanding how Axal implements self-custody for maximum security.

## What is Self-Custody?

Self-custody means you maintain full control over your private keys and assets.

## Our Self-Custody Approach

- **Private Key Control**: You own your private keys
- **Non-Custodial**: We never hold your assets
- **Secure Storage**: Hardware security modules (HSMs)
- **Backup Options**: Multiple recovery methods

## Wallet Types

- **Hot Wallets**: For active trading and transactions
- **Cold Storage**: For long-term asset holding
- **Multi-Sig**: Enhanced security with multiple signatures

## Security Features

- **Hardware Security**: Enterprise-grade HSMs
- **Encryption**: AES-256 encryption at rest
- **Access Controls**: Multi-factor authentication
- **Monitoring**: 24/7 security monitoring

## Recovery Options

- **Seed Phrases**: 12-24 word recovery phrases
- **Social Recovery**: Trusted contacts can help recover
- **Hardware Backup**: Physical security devices
